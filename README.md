TITLE:-
Olympic Athlete Analysis & Insights Dashboard

PURPOSE:-
The dashboard provides interactive insights into how athletes perform over time, which countries dominate the Olympics, and how factors like age and gender influence outcomes.

TECH STACK:-
Visualization Tool: Microsoft Power BI
Data Transformation: Power Query (ETL)
Data Modeling: DAX (Measures & Calculations)
Data Format: Excel / CSV
Optional: SQL (for data extraction)

DATA SOURCE:-
Structured dataset containing athlete, event, medal, age, gender, and country details.

FEATURES:-
📈 Medal Trends Over Time (Year-wise analysis)
🌍 Top 5 Countries by Medals (Comparative insights)
👥 Gender Distribution Analysis
🎂 Age-wise Athlete Participation
🏅 Medal Breakdown (Gold, Silver, Bronze)
❄️☀️ Season-wise Sports Analysis (Summer vs Winter)
🎯 Interactive Filters (City & Year slicers)
📊 KPI Cards (Total sports, participation insights)
